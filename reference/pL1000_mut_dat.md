# Mutation count data for pL1000_200_1

Long-format data frame of mitochondrial variant read counts (200 cells,
186 variants, 37 200 rows).

## Usage

``` r
pL1000_mut_dat
```

## Format

A data frame with columns:

- cell:

  Cell barcode identifier.

- variant:

  Variant identifier (e.g. `"10111_T_C"`).

- a:

  Alternate allele read count.

- d:

  Total read depth.

## Source

pL1000_200_1 dataset.
