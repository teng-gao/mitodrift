# Annotated phylogenetic tree for pL1000_200_1

A rooted binary `phylo` object (200 tips, 199 internal nodes) with
per-node confidence scores stored in `node.label`. Built from a small in
vitro LARRY barcode sample (pL1000, 200 cells).

## Usage

``` r
pL1000_tree_annot
```

## Format

An [`ape::phylo`](https://rdrr.io/pkg/ape/man/read.tree.html) object.

## Source

MitoDrift inference on the pL1000_200_1 dataset.
